# 🏋️‍♂️ FitTrack Pro – Fitness & Diet Dashboard

A modern, responsive **Fitness & Diet Dashboard** built using React and Tailwind CSS.
This project helps users track their fitness activities, diet, water intake, and overall health progress with a clean SaaS-style UI.

---

## 🚀 Features

### 📊 Dashboard

* Overview of daily activity (steps, calories, goals)
* Weekly analytics charts
* BMI calculator
* Smart meal suggestions
* Water intake tracker (quick view)

### 🏃 Activity Tracking

* Add workouts (steps, calories, duration)
* View activity history
* Default activity list support
* Data stored using localStorage

### 🥗 Diet & Nutrition

* Categorized meal plans:

  * Weight Loss
  * Weight Gain
  * Fat Loss
  * Fat Gain
  * Muscle Gain
* Veg & Non-Veg options
* Smart meal recommendations
* Nutrition breakdown (Calories, Protein, Carbs, Fat)

### 💧 Water Intake Tracker

* Track daily water consumption
* Progress bar with goal tracking
* Quick add buttons (+250ml, +500ml)
* Daily reset logic

### 🏆 Achievements System

* Unlock badges:

  * 7-day streak
  * 10k steps
  * Goal completion

### 🔔 Notifications

* Smart alerts:

  * Missed goals
  * Water reminders

### 🤖 AI Chatbot

* Fitness assistant for:

  * Workout suggestions
  * Diet recommendations

### 👤 Profile Page

* User details (height, weight, goals)
* Editable profile

### ⚙️ Settings Page

* Theme toggle (Dark / Light mode)
* Wearable devices (dummy integration)
* About Us section
* Terms & Conditions
* Developer info card

### 🌙 Dark Mode

* Full app support
* Saved user preference
* Smooth transitions

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Charts:** Recharts
* **State Management:** React Context API
* **Storage:** LocalStorage

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/fitness-dashboard.git
```

2. Navigate to project folder:

```bash
cd fitness-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:5173
```
